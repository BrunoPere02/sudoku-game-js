# 🔢 Sudoku Game

Jogo de Sudoku desenvolvido em JavaScript puro com 3 níveis de dificuldade.

## 📋 Sobre o Projeto

Jogo de Sudoku completo desenvolvido com HTML, CSS e JavaScript puro, com geração automática de tabuleiros válidos e sistema de validação em tempo real.

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3** (animações e design responsivo)
- **JavaScript** (lógica do jogo com Orientação a Objetos)

## 🎮 Funcionalidades

- ✅ Geração automática de tabuleiros válidos
- ✅ 3 níveis de dificuldade (Fácil, Médio, Difícil)
- ✅ Timer com pause/resume
- ✅ Destaque de erros em tempo real
- ✅ Teclado numérico na tela
- ✅ Suporte ao teclado físico
- ✅ Botão de reiniciar partida
- ✅ Modal de vitória com tempo final
- ✅ Design responsivo para mobile

## ▶️ Como Jogar

1. Abra o arquivo `index.html` no navegador
2. Selecione a dificuldade desejada
3. Clique em uma célula vazia
4. Digite o número pelo teclado ou clique nos botões
5. Complete o tabuleiro sem erros para vencer!
